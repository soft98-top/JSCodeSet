## JSCodeSet
本项目汇总一些平时写的JavaScript脚本，用于快速执行一些操作，默认情况下会给出完整代码、自动更新版本、一句话版本。
### 使用方法
- 添加任意网页到收藏夹，修改网址内容为一句话版本的代码，然后直接在对应的网页点击收藏夹中的网页即可运行代码。
